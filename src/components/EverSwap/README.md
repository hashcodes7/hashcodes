## 🌲 Overview

**EverSwap** is a unified DeFi protocol that brings together **decentralised trading**, **permissionless lending**, and **zero-interest borrowing** into a single, capital-efficient system. Built on **single-sided liquidity pools** and powered by **relayers**, EverSwap redefines how liquidity flows through on-chain markets.

> *One Pool. Every Function.*

The protocol is designed for three core user groups — **Liquidity Providers**, **Borrowers**, and **Traders** — each benefiting from a cleaner, deeper, and more efficient liquidity model.

### 🔹 Single-Sided Liquidity Pools
Deposit a single asset into a pool and earn yield from **swap fees**, **borrowing interest**, and **internal arbitrage** — no need to provide balanced pairs.

### 🔹 Relayer-Powered Architecture
A new model that enables deeper, cleaner, and more efficient liquidity distribution across the entire system.

### 🔹 Unified Capital Efficiency
EverSwap unifies trading, lending, and borrowing, so the same liquidity serves multiple functions simultaneously — smoother execution, more efficient markets, and better capital use.

<br/>

## 🧭 For Everyone

### 💧 Liquidity Providers
> *"Deposit one asset and earn from swaps, borrowing, and internal arbitrage."*

Contribute to a pool with a single asset and automatically capture revenue from multiple protocol activities. No complex LP management required.

### 📋 Borrowers
> *"Access capital through single-asset collateral and a borrowing model built for changing markets."*

Use a single asset as collateral to borrow against — a flexible model designed to adapt to evolving market conditions with zero-interest borrowing.

### 📈 Traders
> *"Get better execution with direct swaps, cleaner routing, and more efficient markets."*

Trade directly against concentrated liquidity pools with minimal slippage and more predictable execution.

<br/>

## 🧱 One Pool, Every Function

EverSwap's core innovation is its **unified pool architecture**. Instead of siloed liquidity for trading, lending, and borrowing, a single pool serves all three functions:

```
┌─────────────────────────────────────────────────────┐
│                   Single Pool                        │
├─────────────┬─────────────────┬─────────────────────┤
│    Swap     │     Lend        │      Borrow         │
│  Engine     │    Engine       │     Engine          │
├─────────────┼─────────────────┼─────────────────────┤
│   Direct    │  Earn yield     │  Single-asset       │
│   swaps     │  from fees      │  collateral model   │
│   Routing   │  + arbitrage    │  Zero-interest      │
└─────────────┴─────────────────┴─────────────────────┘
```

This convergence creates **smoother execution**, **more efficient markets**, and **better capital use** across the system.

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/everswap/everswap.git
cd everswap

# Install dependencies (if using a package manager)
# (The project is currently a static HTML/CSS/JS build)

# Serve locally
npx serve .
# or
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
